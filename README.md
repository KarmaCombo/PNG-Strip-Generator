# PNG Strip Generator

A web-based tool that converts GIF animations into vertical PNG strips for use in Minecraft texture packs.

## Features

- **Drag & Drop Support** - Easy GIF upload interface
- **Frame Extraction** - Extracts all frames from animated GIFs
- **Vertical PNG Strip** - Combines frames into a single vertical strip
- **Resize Options** - Scale to 16x, 32x, 64x, or keep original size
- **Frame Time Control** - Adjust animation timing
- **ZIP Package** - Generates ready-to-use texture package
- **Live Preview** - See animation preview before downloading

## Usage

1. Open `PNG Strip Generator.html` in your web browser
2. Upload or drag & drop a GIF file
3. Choose resize option and frame time settings
4. Click "Generate & Package ZIP"
5. Download the PNG strip or complete ZIP package
